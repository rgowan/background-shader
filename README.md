# Full page background shader
A simple coloured background shader using CSS3.


## Selector
```
body:before {
  content: '';
  position: absolute;
  top: 0;
  left: 0;
  width: inherit;
  height: inherit;
  background: rgba(130, 39, 146, 0.75);
}
```
